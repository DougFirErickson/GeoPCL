About

Jul 8th, 2012

GeoPCL is a collection of geospatial extensions to the Point Cloud Library (PCL).

Features

Conversions

Conversions currently only consist of XYZ, and intensity fields if they exist.

  • LAS to/from PCD
  • PDAL-supported formats to PCD

IDL Bindings

  • LAS to PCD
  • Convex hull

Raster Support

  • Gridding

    1. Native implementation
    2. Linking against points2grid library

Future

  • Bare earth extraction

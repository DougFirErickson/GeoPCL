About

Jul 8th, 2012

GeoPCL is a collection of geospatial extensions to the Point Cloud Library (PCL). GeoPCL depends on PCL, as well as libLAS, PDAL, and points2grid. To build the IDL bindings, one must have IDL installed.

GeoPCL can open point cloud data in the LAS format (XYZ and intensity fields only) and convert it to the PCD format used internally by PCL. PDAL supported formats can be converted as well. Once the data has been processed by PCL, it can then be converted back to LAS for export.

GeoPCL’s IDL bindings are intended to be used to expose PCL modules to IDL users. There are currently two IDL bindings, an LAS to PCD conversion example and a convex hull calculation.

GeoPCL has a native implementation of a gridding algorithm and also links to the point2grid library for more advanced gridding approaches (e.g., min, max, average, IDW, …).

Future releases of GeoPCL will extend the ability to perform common geospatial processing tasks for point cloud data, such as feature extraction and ground return filtering.
